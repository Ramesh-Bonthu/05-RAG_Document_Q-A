# 📚 RAG Document Q&A with Groq and Hugging Face

An interactive **Retrieval-Augmented Generation (RAG)** application built using **Streamlit**, **LangChain**, **Groq LLM**, and **Hugging Face embeddings** to perform question answering over research papers.

---

## 🚀 Features

- 🔍 Ask questions from research documents
- 📄 PDF document loading and processing
- ✂️ Smart text chunking using Recursive Character Splitter
- 🧠 Embedding generation using Hugging Face models
- 📦 Vector storage using FAISS
- ⚡ Fast inference using Groq LLM
- 🌐 Interactive UI with Streamlit

---

## 🏗️ Tech Stack

- **Frontend/UI**: Streamlit  
- **LLM**: Groq (`openai/gpt-oss-120b`)  
- **Embeddings**: Hugging Face (`all-MiniLM-L6-v2`)  
- **Vector Store**: FAISS  
- **Framework**: LangChain

# 📚 Conversational RAG Chain with PDF's uploads and Chat History

An advanced **Retrieval-Augmented Generation (RAG)** application that allows users to upload PDFs and interact with them conversationally using **chat history-aware context**.

Built with **Streamlit, LangChain, Groq LLM, and Hugging Face embeddings**.

---

## 🚀 Features

- 📄 Upload multiple PDF documents
- 💬 Conversational Q&A with memory (chat history)
- 🧠 Context-aware responses using history-aware retriever
- 🔍 Semantic search using embeddings
- ⚡ Fast responses using Groq LLM
- 📦 Vector storage using FAISS
- 🖥️ Interactive UI with Streamlit

---

## 🏗️ Tech Stack

- **Frontend/UI**: Streamlit  
- **LLM**: Groq (`openai/gpt-oss-120b`)  
- **Embeddings**: Hugging Face (`all-MiniLM-L6-v2`)  
- **Vector Database**: FAISS  
- **Framework**: LangChain
